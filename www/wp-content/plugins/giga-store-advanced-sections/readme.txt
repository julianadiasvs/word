Add Slider, Testimonials, Banner sections to Giga Store WordPress theme.

== Description ==

Giga Store Advanced Sections creates advanced sections for WooCommerce WordPress theme Giga Store by themes4wp.com. Add Slider, Testimonials, Banner sections to Giga Store WordPress theme. This plugin won't do anything for other free or premium WordPress themes and you need to download and install <a href="https://wordpress.org/themes/giga-store/" target="_blank">Giga Store</a>.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the whole contents of the folder `giga-store-advanced-sections` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress dashboard
3. Enjoy using it :)

== Frequently Asked Questions ==

= What themes this plugin supports? =

Currently it works only with Giga Store theme.

= Am I obligated to use it? =

You can still use Giga Store theme without this plugin but you won't be able to setup some homepage sections.
